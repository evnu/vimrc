vimrc
=====

My vim dotfiles. That's it.

Requirements
============

* pathogen
* vundle
* [fzf](https://github.com/junegunn/fzf)

Installation
============

* Clone the repository to `~/.vim` or symlink the clone
* Run
        git submodule init
        git submodule update
* Then run `:PluginInstall`.
